Projeto 1: Website Responsivo

Detalhes do Projeto

Este projeto foi criado para demonstrar a capacidade de criar sites totalmente responsivos utilizando HTML5 e CSS3. O layout é adaptável a diferentes resoluções de tela, garantindo uma experiência fluida em qualquer dispositivo.

Tecnologias utilizadas: HTML5, CSS3, Flexbox.

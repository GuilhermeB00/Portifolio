Projeto 1: Website Responsivo

Detalhes do Projeto

Este projeto foi criado para demonstrar a capacidade de criar sites totalmente responsivos utilizando HTML5, CSS3 e Flexbox. O design se adapta a diferentes tamanhos de tela, garantindo uma boa experiência do usuário em smartphones, tablets e desktops.

Tecnologias: HTML5, CSS3, Flexbox

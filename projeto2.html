Sistema de Login

Um sistema de login básico desenvolvido para validar usuários e permitir o acesso a áreas restritas. Criado utilizando HTML, CSS e JavaScript, o projeto conta com verificações simples e feedback visual para o usuário.

  • Validação simples com JavaScript
  • Design limpo e responsivo
  • Feedback de erros
  • Segurança básica com hash de senhas

Ideal para pequenas aplicações web, o sistema de login oferece uma interface amigável e funcional para proteger páginas e seções privadas.
